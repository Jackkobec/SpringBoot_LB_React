# How to start:
### Для FrontEnd корня проекта запускаем Terminal:
#### cd ui
#### npm i
#### npm start

### FrontEnd запускается  на localhost/3004

### Для BackEnd запускаем еще один Terminal:
#### mvn clean  (не обязательно, но жетательно)
#### mvn spring-boot:run

### FrontEnd запускается  на localhost/8080
